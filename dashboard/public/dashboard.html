About Active Developer Badge

Discord requires developers to actively use at least one slash command within 60 days to maintain Active Developer status.

  • Automatic command execution every 30 days
  • Registers commands on startup
  • Auto-reconnect on disconnection
  • Works in servers, DMs, and group DMs
Tip: Use /status in Discord to check next execution date

Moderation Tools

Comprehensive moderation commands to manage your server. All commands respect Discord permissions.

Member Actions
/kick <user> [reason]

Remove a user from the server

/ban <user> [reason]

Permanently ban a user

/mute <user> <minutes> [reason]

Timeout a user temporarily

/unmute <user>

Remove timeout from a user

/warn <user> [reason]

Issue a warning to a user

Channel Management
/lock

Prevent members from sending messages

/unlock

Allow members to send messages again

/slowmode <seconds>

Set delay between messages (0-21600s)

/purge [amount]

Bulk delete messages (up to 1000, max 14 days old)
