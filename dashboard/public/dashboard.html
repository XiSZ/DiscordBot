About Active Developer Badge

Discord requires developers to actively use at least one slash command within 60 days to maintain Active Developer status.

  • Automatic command execution every 30 days
  • Registers commands on startup
  • Auto-reconnect on disconnection
  • Works in servers, DMs, and group DMs
Tip: Use /status in Discord to check next execution date

Moderation Tools

Comprehensive moderation commands to manage your server. All commands respect Discord permissions and role hierarchy.

Member Actions
/kick <user> [reason]

Remove a user from the server

Requires: Kick Members
/ban <user> [reason]

Permanently ban a user

Requires: Ban Members
/mute <user> <minutes> [reason]

Timeout a user temporarily

Requires: Moderate Members
/unmute <user>

Remove timeout from a user

Requires: Moderate Members
/warn <user> [reason]

Issue a warning to a user

Requires: Moderate Members
Channel Management
/lock

Prevent members from sending messages

Requires: Manage Channels
/unlock

Allow members to send messages again

Requires: Manage Channels
/slowmode <seconds>

Set delay between messages (0-21600s)

Requires: Manage Channels
/purge [amount]

Bulk delete messages (up to 1000, max 14 days old)

Requires: Manage Messages
Important Notes
Role Hierarchy

Bot cannot moderate users with roles higher than or equal to its highest role

Message Age Limit

Purge command only works on messages less than 14 days old (Discord limitation)

Permission Check

Both you and the bot must have the required permission for each command

Timeout Duration

Mute duration can be up to 40,320 minutes (28 days)
